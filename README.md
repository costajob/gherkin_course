## Bank Account
I created a plain simple bank account class to test balance, deposit, withdraw and PIN reset functionalities.

## Usage
Install dependencies by using bundler:
```
bundle install
```

Once done run Cucumber specs by:
```
cucumber
```

Run unit tests by:
```
rake
```
